# File Manager Project

This project is a simple file manager application built using Python and Tkinter. It provides a graphical interface to perform basic file system operations.

## Features

- Create files and directories  
- View file contents  
- Edit and save files  
- Delete files  
- Rename files  
- Navigate between directories  
- View file metadata such as size and last modified time  

## How to Run

1. Open terminal  
2. Navigate to the project folder  
3. Run the following command:
```
python main.py

```

## Requirements

- Python 3.x  
- Tkinter (comes with Python)
